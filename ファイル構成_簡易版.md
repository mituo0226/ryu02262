```
kaede/
├── nocss/                        # CSSなし版フォルダ
│   ├── index.html                # メインファイル（インラインスタイルのみ）
│   └── index.backup.html         # バックアップ
│
├── public/                       # メイン制作フォルダ（Cloudflare公開用）
│   ├── index.html                # CSSあり版（メインファイル）
│   ├── index.backup.html         # バックアップ
│   ├── styles.css                # スタイルシート
│   ├── styles.backup.css         # バックアップ
│   └── photo/                    # 画像フォルダ
│       ├── rogo.png              # サイトロゴ
│       ├── kaede.png             # 楓のプロフィール写真
│       ├── kaon.png              # 三崎花音のプロフィール写真
│       ├── yukino.png            # 笹岡雪乃のプロフィール写真
│       └── sora.png              # 水野ソラのプロフィール写真
│
├── index.html                    # ルート版（CSSあり、public/を参照）
├── .cursorrules                  # Cursor AI向けルール定義
├── README.md                     # プロジェクト説明書
├── プロフィール.txt               # 鑑定士のプロフィール情報
└── プロジェクト構成図.txt         # 詳細構成図
```

## 制作ルール

**制作順序：**
1. `public/index.html` で制作（CSSあり版）← 先にこちら
2. `nocss/index.html` で制作（CSSなし版）← その後

**nocss/index.html の制約：**
- `<style>`タグ禁止
- 外部CSSファイル参照禁止
- すべてインラインスタイルのみ

**画像パス：**
- `public/index.html` → `photo/`
- `nocss/index.html` → `../public/photo/`
- `index.html` (ルート) → `public/photo/`
